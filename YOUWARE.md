# EVLogistics Platform

A modern, high-performance landing page for a nationwide EV logistics infrastructure platform.

## Project Status

- **Project Type**: React + TypeScript Modern Web Application
- **Entry Point**: `src/main.tsx`
- **Build System**: Vite 7.0.0
- **Styling System**: Tailwind CSS 3.4.17

## Features Implemented

1.  **Modern Landing Page**:
    -   **Hero Section**: Full-screen immersive background with animated text and CTA.
    -   **Stats Section**: Key performance indicators with animated counters.
    -   **Features Section**: Split-layout showcasing "Complete Operational Visibility" and "Scalable Charging Infrastructure".
    -   **Footer**: Comprehensive footer with CTA and links.

2.  **UI Components**:
    -   `Button`: Reusable button component with variants (primary, secondary, outline, ghost).
    -   `Navbar`: Responsive navigation bar with scroll effects.

3.  **Assets**:
    -   Custom AI-generated images for Hero, Dashboard, Charging, and CTA background.
    -   Lucide React icons for visual communication.

## Tech Stack

-   **Framework**: React 18
-   **Language**: TypeScript
-   **Styling**: Tailwind CSS
-   **Animation**: Framer Motion
-   **Icons**: Lucide React
-   **Utilities**: clsx, tailwind-merge

## Development

To run the project locally:

```bash
npm install
npm run dev
```

To build for production:

```bash
npm run build
```
